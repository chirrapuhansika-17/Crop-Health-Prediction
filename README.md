# Crop Health Prediction 🌱

A machine learning-based web application that predicts crop health conditions using deep learning techniques. The system helps identify crop diseases and health issues from plant images, enabling farmers and users to take preventive actions early.

## 🚀 Features

- 🌿 Crop health prediction using Deep Learning
- 🧠 CNN-based image classification model
- ⚡ Real-time prediction system
- 🌐 User-friendly web interface
- 🔗 Integrated frontend and backend architecture
- 📊 Accurate disease detection and classification

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- FastAPI

### Machine Learning
- Python
- Convolutional Neural Network (CNN)
- TensorFlow / Keras
